<h1>Preguntas Frecuentes ❓</h1>

<p>A continuación, encontrarás respuestas a las preguntas más comunes sobre el proyecto <strong>React Game App for Beginners</strong>.</p>

<h2>🧐 ¿Por qué mi aplicación no se ejecuta en Android/iOS?</h2>
<p>Asegúrate de que todos los requisitos previos estén instalados, como Android Studio o Xcode. Verifica que el emulador esté ejecutándose antes de usar los comandos <code>run-android</code> o <code>run-ios</code>.</p>

<h2>🛠️ ¿Cómo puedo resolver errores de dependencias?</h2>
<p>Ejecuta el siguiente comando para limpiar la caché:
<pre><code>npx react-native start --reset-cache</code></pre>
</p>
